📌 Key facts about .find():
• Returns the index position of the first match
• Returns -1 if the substring is not found
• It is case-sensitive — "cat" ≠ "Cat"
• Syntax: position = my_string.find("search_word")

🔧 Using Subroutines (Functions) ACTIVITY 5
📌 What is a subroutine?
A subroutine (also called a function) is a reusable block of code that performs a specific task. In Python, we create subroutines using the def keyword. This helps us:
• Avoid repeating code — write once, use many times
• Organise our program — break it into smaller, named blocks
• Make code easier to read — each function has a clear purpose
Python — using subroutines 
# Define a subroutine to search for a word in a line
def search_line(line, keyword):
    position = line.lower().find(keyword.lower())
    if position != -1:
        print(f"Found '{keyword}' at position {position} in: {line}")
        return True
    else:
        return False

# Main program — uses the subroutine
input_file = open("input.txt", "r")
count = 0

for line in input_file:
    line = line.strip()
    if search_line(line, "Jersey"):
        count += 1

print(f"\nTotal matches: {count}")
input_file.close()
🔑 Key points about subroutines:
• def search_line(line, keyword): — defines a function with parameters
• return True — sends a value back to where the function was called
• search_line(line, "Jersey") — calls the function with arguments
• The function can be called many times with different keywords!
